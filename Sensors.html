--- SENSORS & MODULES ---


What are Sensors and Modules

Sensors and modules (having extra electronic circuitry along with sensor) are Electronic devices that detect and respond to some type of input from the physical environment.Thease Sensors and Modules can be used for any Electronic project in addition to electronic platfroms like Arduino,Nodemcu, and Rasberrypies. Here we are talking about a large number of sensors and Modules but not all of them.

Sensors

Sensors are defined as a machine, module, or device that detect changes in the environment. The sensors transfer those changes to electronic devices in the form of a signal. The output signal is easily readable by humans and is an important part of our daily lives.

And the sensor is a device, which is made up of Single Crystal Silicon. It has superior mechanical stability, machinability, etc. It can also combine electronics and sensing elements on the same substrate. The data signal runs from the sensor to the output pins of the Arduino.

PIR Sensor
  • Passive Infrared Sensor or simply PIR Sensor allows you to detect or sense any motion. A PIR Sensor Module like HC-SR501 is a commonly used modules in arduino projects. Using a PIR Sensor Module, you can implement Motion Detection Applications using Arduino.
Water Level Sensor
  • Water level sensors (or leak sensors) allow you to gauge the presence of water or the depth of water by reading the sensor output values. Great for detecting leaks in the pans of hot water heaters.
Flame IR Sensor
  • The KY-026 IR Flame sensor module detects the infrared light spectrum emitted by open flames and outputs a digital signal.
Heart Beat/Pulse Sensor
  • A human finger is placed between the infrared diode and the photo transistor in the KY-039 heartbeat sensor to detect a pulse. The signal output pin will be used to represent the pulse. This sensor detects the presence of light, in this case how much light is going through a finger, using a photo transistor. The amount of light changes when blood moves, and this shift can be sensed as a pulse.
High Sensitivity Microphone Sensor
  • The KY-037 sensor is a sound and audio detection module with a high sensitivity. Based on the sound intensity, this sensor generates a digital output. Based on the position of an on-board potentiometer, the trigger value can be modified.
Infrared(IR) Sensor
  • The KY-022 has three pins and operates at 940nm with a carrier frequency of 38kHz. The digital output receives this signal. If an IR signal is detected, an LED module on the board will flash. This sensor module is ideal for decoding remote controllers for home audio/visual equipment (home theater) and other IR-based devices.
Knock Sensor
  • Knock sensors are vibration sensors that are specifically intended to detect knocking vibrations. This is ideal for magic door openers that are activated by knocking on the door.
Metal Touch Sensor
  • The KY-036 sensor module is used to detect human contact. It's possible to utilize it to control projects depending on human or animal contact with a metal-connected device. It's ideal for building a touch lamp.
Ultrasonic Sensor
  • Ultrasonic distance sensors, such as the KY-050, send out a high-frequency ultrasonic pulse that is reflected by the on-board ultrasonic sensor. The time delay can be used to calculate distances with amazing precision. 
Soil Moisture Sensor
  • Soil moisture sensors let you accurately detect the amount of water suspended in soils (plant pots for example). These are great for Arduino gardening projects.
Temparature Sensor
  • The KY-013 Analog Temperature Sensor monitors ambient temperature with an accuracy of 0.5°C between -55°C and +125°C. This is typically used to monitor the temperature of your Arduino project's devices and enclosures. 
Tracking Sensor
  • Tracking sensor are used to help robots follow a specific path on paper or other surfaces by sensing lines and markings drawn on their surfaces.
Vibration Sensor
  • The KY-002 vibration switch module detects and connects two pins to detect movement and vibration in your project (digital output). This may be used to detect whether or not your Arduino project is being shaken or relocated abruptly. 

This is not a complete Sensor list.There are many More..

Modules

In the Arduino environment, a "Module" is a ready-to-use board with a Sensor or an IO device on it. A few additional components are required to use a sensor or device (such as an LCD). An Arduino Module incorporates all of the essential components and connections for a sensor (or a device) to function and provides you with a ready-to-use product. Consider the following module, which is both basic and useful: The Infrared Obstacle Avoidance Sensor Module is a sensor that detects obstacles in the infrared spectrum. It comprises of an IR LED, a Photo Diode, an Op-Amp, a Potentiometer, a couple of passive components, and a connector. The "Sensor" side of the circuit is taken care of by the module.

LCD Display Module
  • The LCD (Liquid Crystal Display) is a form of display that operates by using liquid crystals. We'll take serial input from the computer and upload the sketch to the Arduino from here. On the LCD, the characters will be shown. Liquid Crystal Library is the library that allows us to control the LCD display. It may operate in either an 8-bit or 4-bit mode. The bit mode here refers to the data lines as well as the enable, rs, and rw control lines.
GSM Module
  • A GSM Module is essentially a GSM Modem (such as the SIM 900) attached to a PCB with various sorts of outputs, such as TTL Output (for Arduino, 8051, and other microcontrollers) and RS232 Output (to communicate directly with a computer) (personal computer). The board will also contain pins or provisions for connecting a microphone and speaker, as well as for disconnecting +5V or other power and ground connections. Different modules have different types of provisions. There are many different types of GSM modems and GSM Modules to pick from on the market. It's usually best to use an arduino compatible GSM Module for our project of connecting a gsm modem or module to arduino and therefore sending and receiving sms using arduino.
GPS Module
  • The Global Positioning System (GPS) uses signals from orbiting satellites and ground stations on Earth to identify its precise location on the planet. To interface with a microcontroller or a PC terminal, the NEO-6M GPS receiver module uses USART communication. It gets information from satellites in the form of NMEA strings, such as latitude, longitude, altitude, UTC time, and so on. To extract the information we want to use, we must parse this string. 
Gas Sensor Module
  • Often used in safety related applications, the MQ2 Gas Sensor Module is used to detect Smoke, Carbon Monoxide, Hydrogen and other gases.Use this MQ2 Sensor Module to implement Smoke Detection Alarm using Arduino.
Blutooth Module
  • Bluetooth is a wireless communication standard for short-range communication. It operates in the 2.4–2.485 GHz (unlicensed) frequency spectrum and is utilized for low-power, low-cost wireless data transfer applications.
Keypad Module
  • Keypad is an input device which is generally used in applications such as calculator, ATM machines, computer etc.
ESP8266 Wifi Module
  • ESP8266 is Wi-Fi enabled system on chip (SoC) module developed by Espressif system. It is mostly used for development of IoT (Internet of Things) embedded applications.
Real Time Clock Module
  • DS1307 is a two wire (I2C) serial interface RTC (Real Time Clock) with 56 bytes of nonvolatile RAM. This provides clock and calendar with second, minute, hour, day, date, month and year.
